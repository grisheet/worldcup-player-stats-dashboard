# ⚽ World Cup Player Stats Dashboard

A polished FIFA World Cup Player Stats Dashboard built with **HTML**, **CSS**, **JavaScript**, and **Chart.js** — featuring historical data visualizations, KPI cards, dark mode, and interactive charts.

## 🎖️ Live Demo

**[View Dashboard →](https://grisheet.github.io/worldcup-player-stats-dashboard/)**

## 📊 Features

### Interactive Visualizations
- 🏆 **Top World Cup Scorers** — Bar chart of all-time leading goal scorers
- 📊 **Goals Per Tournament** — Line chart tracking goals across World Cup editions (1930-2022)
- ⏱️ **Goals by Match Period** — Doughnut chart showing when goals are scored
- 👥 **Most Capped Players** — Players with most World Cup appearances
- 🟡🔴 **Discipline Trends** — Yellow and red cards per tournament
- 🎯 **Average Goals Per Match** — Historical trends in scoring rates

### Dashboard Components
- 📊 **KPI Cards** — Quick stats on tournaments, matches, goals, players, nations
- 🔍 **Filters** — Filter by tournament year, country, and stage
- 🌗 **Dark/Light Mode** — Seamless theme toggle with localStorage persistence
- 🎭 **Player Comparison** — Side-by-side stats comparison tool
- 📋 **Data Table** — Searchable table of top players
- ✨ **Insights** — Auto-generated highlight badges

### Design
- 🎨 Premium sports analytics aesthetic
- 🟢 Color palette: Deep green, off-white, charcoal, gold accents
- 📱 Fully responsive design
- ⚡ Smooth animations and hover states

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Charts**: Chart.js v4.4.0
- **Fonts**: Google Fonts (Inter)
- **Data**: Embedded historical World Cup statistics (1930-2022)

## 🚀 Getting Started

### Option 1: View Live
Visit **[https://grisheet.github.io/worldcup-player-stats-dashboard/](https://grisheet.github.io/worldcup-player-stats-dashboard/)**

### Option 2: Run Locally

1. Clone the repository:
```bash
git clone https://github.com/grisheet/worldcup-player-stats-dashboard.git
```

2. Navigate to the project folder:
```bash
cd worldcup-player-stats-dashboard
```

3. Open `index.html` in your browser:
```bash
open index.html
```

No build tools or dependencies required!

- Goals per tournament edition

- Match-level statistics
